## 📌 Проекты

### [bankruptcy-analyzer](bankruptcy-analyzer/index.html)
Анализатор банкротных рисков контрагентов из Excel-выгрузки «События мониторинга». Извлекает наименования и даты событий, присваивает каждому событию уровень риска (высокий / средний / низкий) по словарю признаков несостоятельности и группе событий. Работает офлайн — файл обрабатывается в браузере и не покидает устройство. Есть выгрузка результата в CSV. Инструмент для службы экономической безопасности.
